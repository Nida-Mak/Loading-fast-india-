# Loding Fast India — Transport Management App

## Overview
A full-featured logistics and transport management mobile app with separate Driver and Merchant flows, bilty (lorry receipt) generation, GST compliance, commission management, real-time chat, and AI-powered distance-based fare calculation.

## GST Details
- **GSTIN**: 24BRLPS3959R1ZN
- **GST on Transport**: 5% IGST
- **Driver Commission**: 5% of freight amount

## App Structure

### Screens
- `app/index.tsx` — Role selection (Driver / Merchant)
- `app/login.tsx` — Login with email (role-specific)
- `app/register.tsx` — Registration (role-specific)
- `app/merchant/index.tsx` — Merchant dashboard + trip list
- `app/merchant/book.tsx` — 3-step booking with auto fare calculator
- `app/merchant/bilty.tsx` — Bilty / Lorry Receipt viewer + chat
- `app/driver/index.tsx` — Driver dashboard + available trips
- `app/driver/trip.tsx` — Trip detail + action buttons + chat
- `app/chat.tsx` — Real-time chat between driver and merchant

## Database (PostgreSQL)
- **Table**: `lfi_users` — stores all driver and merchant registrations permanently
- Users are saved to the database on registration, survives server restarts
- Admin panel reads users from the database (all devices, real-time)
- Suspend/unsuspend actions also update the database
- Driver ratings update the database

### Context/State
- `context/auth.tsx` — User authentication (AsyncStorage)
- `context/trips.tsx` — Trip CRUD operations (AsyncStorage)

### Backend API (Express on port 5000)
- `GET /api/fuel-prices` — Current diesel/petrol/CNG prices
- `POST /api/fuel-prices` — Update prices
- `POST /api/calculate-fare` — Auto fare calculation (distance + fuel + vehicle + weight)
- `GET /api/messages/:tripId` — Get chat messages for a trip
- `POST /api/messages` — Send a chat message

### Constants
- `constants/vehicles.ts` — Vehicle types with capacity and fuel efficiency
- `constants/colors.ts` — Navy + Orange theme colors

- 10 Wheeler Truck (20T) — 5 km/L
- Trailer / Container (40T) — 3.5 km/L

## Fare Calculation Formula
`Total Fare = Fuel Cost + Toll + Driver Wages + Maintenance + 5%  Commission + 15% Profit`
- Fuel Cost = Distance / Vehicle Efficiency × Current Diesel Price
- Toll: ₹0.8–₹2.5 per km (vehicle-dependent)
- Wages: ₹2.5–₹5 per km
- Maintenance: ₹1.5–₹5 per km
- Platform commission: 5% of base cost
- Profit margin: 15%
- GST 5% added on top

## Features
- Merchant: Register, book trips, distance input, auto fare (fuel-price-adjusted), GST bilty
- Driver: Register with vehicle, browse available trips, accept, confirm, cancel, pay commission
- Chat: Real-time (3s polling) messaging per trip between driver and merchant
- Bilty: Full lorry receipt with GSTIN, GST breakdown, shareable
- All trip data persisted with AsyncStorage (local backup)
- Backend: In-memory store for chat messages + API-based fare calculation
